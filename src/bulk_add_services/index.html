Bulk Add Services

Add multiple services to your FireHydrant account,
or use a CSV file for bulk upload.

Note: For CSV uploads, please ensure proper formatting. View CSV instructions.

Optional: Upload a CSV file to add multiple services

CSV Upload Instructions:

  • name: The name of the service (Required).
  • description: A brief description of the service (Optional).
  • remoteId: Unique identifier for the service from alerting provider (Required).
  • connectionType: Type of service connection, e.g., 'pager_duty', 'victorops', 'opsgenie', 'github' (Required).
  • alertOnAdd: 'true' or 'false' to enable/disable alerts on add (Optional).
  • autoAddRespondingTeam: 'true' or 'false' for auto addition of responding team (Optional).
  • ownerId: ID of the service owner (Optional).
  • teamsId: ID of the associated team (Optional).
  • functionalitiesId: ID of related functionalities from FireHydrant. Can be found here. (Optional).
